# Скрипты для conditions

Скрипты для должны представлять собой модули ansible импортируемые с помощью environment переменной ANSIBLE_LIBRARY (либо аналогичные импортируемые модули для других configuration tools).
И должны быть расположены в папке **{ configuration tool name }_plugins/plugins/modules** в clouni client:

## Ansible conditions

Здесь факты это объект вида ключ-значение, 
где ключи - это параметры облачных ресурсов, а значения - значения этих параметров.

Разработчик модуля для фильтрации фактов должен использовать переменные *input_facts*, *input_args*, первая из которых представляет собой список фактов, 
а вторая аргументы для фильтрации этих фактов. Модуль должен возвращать только один найденный факт или выдавать ошибку с сообщением о том, что подходящих фактов не найдено.

Стандартное поведение в случае нескольких подходящих фактов - *возвращать последний найденный*.
